## Project Concept
NeuraText AI is a Python-based conversational AI assistant that enables real-time intelligent interaction using a large language model API. <br/>
Designed with extensibility in mind, the system supports future integrations such as Telegram, WhatsApp, and web deployment.

**It will include:**
* Free to Use (No subscription)
* Conversational memory
* Configurable personality modes
* Clean UI
* Scalable backend
